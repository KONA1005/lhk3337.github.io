## 리액트 설치

node와 npm을 설치되었다는 가정하에 아래와 같이 리액트를 설치해 주세요

```zsh
$ npx create-react-app todos
```

## 리액트 실행

```zsh
$ cd todos
$ npm start
```

프로젝트 폴더로 이동 후 npm start로 리액트 서버 실행을 합니다.

## 코드

#### src/index.js

```jsx
import React from "react";
import ReactDOM from "react-dom";
import "./index.css";
import App from "./App";

ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById("root")
);
```

#### src/App.js

```jsx
function App() {
  return (
    <div>
      <h1>Hello React</h1>
    </div>
  );
}

export default App;
```
